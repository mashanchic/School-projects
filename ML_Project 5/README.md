## Описание
Изучение деревьев решений (CART) и ансамблевых методов (Random Forest, Gradient Boosting) для задач классификации и регрессии.

**Задача:** Предсказание некачественных автомобилей (IsBadBuy)

**Данные:** https://www.kaggle.com/competitions/DontGetKicked
## Ключевые этапы
- Реализация деревьев решений (CART)
- Построение ансамблевых моделей:
  - Случайный лес (Random Forest)
  - Градиентный бустинг (GBDT)
  - ExtraRandomized Trees
- Сравнение современных библиотек:
  - XGBoost
  - LightGBM  
  - CatBoost
- Анализ переобучения и настройка гиперпараметров
- Оценка качества моделей на временных разбиениях
