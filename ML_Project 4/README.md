## Описание
Введение в задачи классификации и основные алгоритмы машинного обучения для решения задач классификации.

**Задача:** Предсказание некачественных автомобилей (IsBadBuy)
**Данные:** https://www.kaggle.com/competitions/DontGetKicked

## Ключевые этапы
- Реализация алгоритмов классификации:
  - Логистическая регрессия
  - Наивный байесовский классификатор
  - Метод k-ближайших соседей (KNN)
- Расчет метрик качества (Precision, Recall, F1, AUC ROC, AUC PR)
- Создание нелинейных признаков
- Отбор признаков и регуляризация
- Оптимизация гиперпараметров моделей ( Optuna )
