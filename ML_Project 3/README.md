## Описание
Изучение методов валидации, оптимизации гиперпараметров и отбора признаков для улучшения качества моделей машинного обучения.

**Задача:** Предсказание цены аренды квартир.

**Данные:** https://www.kaggle.com/competitions/two-sigma-connect-rental-listing-inquiries/data
## Ключевые этапы
- Реализация методов валидации (K-Fold, Stratified K-Fold, Time Series Split)
- Оптимизация гиперпараметров (Grid Search, Randomized Search, Bayesian Optimization)
- Отбор признаков и сравнение качества моделей с различными подходами
- Анализ важности признаков с помощью SHAP
